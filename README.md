# Node AIMA

JavaScript/Node.js implementation of algorithms from Russell and Norvig's 'Artificial Intelligence: A Modern Approach'.

## Installation

Via npm on Node:

```
npm install nodeaima
```

## Usage

Reference in your program:

```js
var nodeaima = require('nodeaima');
```
